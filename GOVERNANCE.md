# OpenPAKT Governance

OpenPAKT is maintained by the Meisterware organisation.

## Specification Evolution

The specification evolves through versioned releases.

Changes are proposed via GitHub issues and pull requests and are reviewed by maintainers before inclusion.

Maintainers make the final decision on whether a change is accepted into the specification.

## Versioning

OpenPAKT follows **semantic versioning**.

- Major versions introduce breaking changes
- Minor versions introduce new capabilities
- Patch versions clarify documentation

## Reference Implementation

The **[Detektor CLI](https://github.com/Meisterware/detektor)** serves as the reference implementation for the OpenPAKT specification.
